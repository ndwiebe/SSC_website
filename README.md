# Slab Savvy CPA

The website for [Slab Savvy CPA](https://slabsavvycpa.com): accounting and tax help for Canadian sports card collectors and dealers, home of the free Card Collector's Tax Playbook.

## Structure

- `project/` is the Astro site itself (pages, components, blog content)
- `.github/workflows/drip-cron.yml` runs the scheduled job behind the email drip
- `docs/` holds the plans and specs the site was built from

## Develop

```bash
cd project
npm install
npm run dev
```
